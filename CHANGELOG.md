# Outline Search Changelog

## [2.0.0] - {PR_MERGE_DATE}

### Added
- "Starred Documents" command to view and manage starred documents
- "Recent Documents" command to view recently accessed documents
- "Browse Collections" command to navigate collections and their documents
- "Create Document" command to create new documents with collection selection
- View document content within Raycast
- Edit document title and content
- View and add comments to documents
- Browse document revision history
- Create public share links for documents
- Comprehensive document actions (view, edit, star, comments, revisions, share)
- Keyboard shortcuts for all actions

### Changed
- Improved Windows compatibility
- Enhanced document list with more actions and metadata

### Removed
- Removed axios dependency in favor of native fetch API
