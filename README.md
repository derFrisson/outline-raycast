## Setup

To use this extension, you need to [obtain an API key](https://www.getoutline.com/developers#description/authentication) from your Outline instance(s). You can then configure the extension by adding the Outline instances you want to access through Raycast, specifying a unique name, the instance's URL, and the API key.

## Commands

This extension provides six commands:

1. **Search Documents** - Search for documents across some or all of the configured Outline instances. Results appear as you type. If you have multiple instances, you can select which ones to search.

2. **Starred Documents** - View all your starred documents across all instances. Star/unstar documents directly from the list.

3. **Recent Documents** - View recently accessed documents across all instances, sorted by date.

4. **Browse Collections** - Navigate through collections and their documents. Supports multi-instance selection.

5. **Create Document** - Create new documents with instance and collection selection. Opens the created document in your browser.

6. **Manage Instances** - Add, edit, or remove Outline instances.
