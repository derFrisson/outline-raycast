## Configuration

1. **Generate an API Key**:
   - Go to your Outline settings (e.g., `https://outline.yourcompany.com/settings/tokens`)
   - Create a new API key
   - Copy the key

2. **Configure the Extension**:
   - Open Raycast
   - Search for "Outline"
   - Select "Extension Preferences"
   - Enter your Outline instance URL and API key

   1. **Outline URL**: The URL of your Outline instance.
   2. Example: `https://outline.yourcompany.com`
   3. Make sure to include the protocol (`https://`) and no trailing slash

## Requirements

- Raycast (latest version recommended)
- Outline account (cloud or self-hosted)
- Outline API key
